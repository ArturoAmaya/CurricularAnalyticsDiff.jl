Result Print

This is a group of functions that print the results of a curricular diff operation in a human-legible way. There are two flavors: executive summary and pretty print.

Executive summary is a simplified version with little explanation of why the changes have occurred.

Pretty print is a more detailed version that attempts to explain why changes have occurred.

CurricularAnalyticsDiff.executive_summary_course — Function
executive_summary_course(results::Dict{String,Any}, course_name::AbstractString)

Print a simplified version of the results of curricular diff for a course with name course_name.

It prints results for centrality, blocking factor and delay factor. Complexity is a linear combination of blocking and delay (and the scale isn't exactly decided yet) so it is left to the user to infer.

Pretty Print

CurricularAnalyticsDiff.pretty_print_curriculum_results — Function
pretty_print_curriculum_results(curriculum_results::Dict{}, desired_stat::DesiredStat)

Prints detailed results of a curricular diff for a whole curriculum and the desired_stat.

DesiredStat is an enum that can take on:

  • ALL: All the other options at once.
  • CEN: Centrality
  • COM: Complexity (Blocking and Delay factors)
  • BLO: Blocking Factor
  • DEL: Delay Factor
  • PRE: Prerequisite Chain
